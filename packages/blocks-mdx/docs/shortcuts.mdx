# Shortcuts

In addition to the toolbar at the top of the editor, Blocks exposes a robust
collection of keyboard and Markdown shortcuts.

## Block-level

Block level styling is applied by beginning a newline with a Block shortcut. They can
also be used to toggle on/off a Block type.

Type      | Markdown Equivalent | Keyboard         | Syntax
--------- | ------------------- | ---------------- | ------
Heading 1 | `# Heading 1`       | <kbd>⌘ ⌥ 1</kbd> | <kbd>#</kbd> + <kbd>space</kbd>
Heading 2 | `## Heading 2`      | <kbd>⌘ ⌥ 2</kbd> | <kbd>##</kbd> + <kbd>space</kbd>
Heading 3 | `### Heading 3`     | N/A              | <kbd>###</kbd> + <kbd>space</kbd>
Heading 4 | `#### Heading 4`    | N/A              |<kbd>####</kbd> + <kbd>space</kbd>
Heading 5 | `##### Heading 5`   | N/A              | <kbd>#####</kbd> + <kbd>space</kbd>
Heading 6 | `###### Heading 6`  | N/A              | <kbd>######</kbd> + <kbd>space</kbd>
Blockqote | `> Here's a quote`  | <kbd>⌘ ⌥ Q</kbd> | <kbd>&gt;</kbd> + <kbd>space</kbd>

## Inline

When editing content in a Block you can apply styling to a subset by wrapping a
portion of the content or even the whole line. This is useful for making items
bold, italic, or highlighting some text.

Type     | Markdown Equivalent | Keyboard                    | Syntax
-------- | ------------------- | --------------------------- | ------
**bold** | `*bold*`            | <kbd>⌘</kbd> + <kbd>b</kbd> | `*text*`
*italic* | `_italic_`          | <kbd>⌘</kbd> + <kbd>i</kbd> | `_text_`
`code`   | `` `code` ``        | None                        | `` `text` ``
