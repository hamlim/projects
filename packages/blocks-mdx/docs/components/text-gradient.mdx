import { TextGradient } from '@blocks/kit'

# TextGradient

The TextGradient is a tiny little component to set a text gradient. You
can also optionally pass it `random={true}` to get a new gradient each time
it renders.

<TextGradient start='#21D4FD' end='#B721FF'>
  <h1>Hello, world!</h1>
</TextGradient>

```js react-live=true
<TextGradient start='#21D4FD' end='#B721FF'>
  <h1>Hello, world!</h1>
</TextGradient>
```

## Usage

```js
import { TextGradient } from '@blocks/kit'
```
